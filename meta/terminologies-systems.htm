Named Systems List 1.12.1

The following names (URIs) may be used in the system element of the Identifier Coding or CodeableConcept data types. If a URI is defined here, it must be used in preference to other identifying mechanisms such as OIDs found in the HL7 OID registry. Other URIs may be used for concepts not listed in this table.

URI Source Comment
Code Systems
http://snomed.info/id SNOMED-CT: (IHTSDO) Concept IDs and Snomed Legacy codes This identifies a code defined in the SNOMED-CT ecosystem managed by IHTSDO. Note that SNOMED-CT is a distributed code systems, and there is no single distribution that contains all defined Snomed-CT codes in all contexts of use.
OID=2.16.840.1.113883.6.96
http://loinc.org LOINC (LOINC.org)

LOINC codes are not case sensitive (LOINC defines some codes with alphabetic characters in them)

OID=2.16.840.1.113883.6.1.
http://unitsofmeasure.org UCUM: (UnitsOfMeasure.org) Case Sensitive Codes Generally used in the Quantity Data Type
OID=2.16.840.1.113883.6.8
http://hl7.org/fhir/sid/icd-10 ICD-10 International (WHO)
OID=2.16.840.1.113883.6.3
http://hl7.org/fhir/sid/icd-9 ICD-9 USA (CDC)
OID=2.16.840.1.113883.6.42
http://hl7.org/fhir/v2/[X](/v) Version 2 table code [X] is the 4 digit identifier for a table. e.g. http://hl7.org/fhir/v2/0203
Only some tables may be treated in this fashion. For some tables, the meaning of the code is version dependent, and so additional information must be included in the namespace, e.g. http://hl7.org/fhir/v2/0123/2.3+, as defined in the v2 table namespace list. Version 2 codes are case sensitive.
OID=2.16.840.1.113883.12.[X] (note that this is the table OID, which is not the same as the codeSystem OID)

Note: This namespace may be revised in the future
http://hl7.org/fhir/v3/[X] A Version 3 code system, from the list of code systems defined here [X] is the code system name. e.g. http://hl7.org/fhir/v3/GenderStatus
OID=[as specified by the index]. Version 3 code systems are case sensitive.

Note: This namespace may be revised in the future
http://www.whocc.no/atc Anatomical Therapeutic Chemical Classification System (WHO)
OID=2.16.840.1.113883.6.73
urn:std:iso:11073:10101 Medical Device Codes defined in ISO 11073-10101 (see RTM Management service)
The ISO/IEEE 11073-10201 Terminology Code is used in the code element, and the Terminology Reference ID is used in the display element (CODE10 and REFID in the reference)
OID=2.16.840.1.113883.6.24
http://nema.org/dicom/dicm DICOM Code Definitions (DCM)
the meanings of codes defined in DICOM, either explicitly or by reference to another part of DICOM or an external reference document or standard
v2/Dicom code="DCM". OID=1.2.840.10008.2.16.4
Identifier Systems
urn:ietf:rfc:3986 URIs (W3C) As defined by RFC 3986 (with many schemes defined in many RFCs). OIDs and UUIDs are represented in their URN form (urn:oid: and urn:uuid: - plain oid: and uuid: are not used). Note that UUIDs are always lowercase in a IRI
No OID
http://hl7.org/fhir/sid/us-ssn SSN (W3C) SSN for USA. Represented with dashes removed
OID=2.16.840.1.113883.4.1
