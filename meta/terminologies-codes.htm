Codes Defined by FHIR v0.11 1.12.3.0

This table contains a list of all the namespaces that define codes in the FHIR specification. When represented in a Coding data type, the system element contains the namespace, and the code element contains the defined code. Note that all the codes in these code systems are case sensitive and must be used the case provided in this specification (usually lowercase).

Note that some of these codes are defined in value sets that also include codes from other code systems as well.
