Resource DeviceObservation - Formal Definitions 4.22.4

Formal definitions for the elements in the deviceobservation resource.

DeviceObservation
DefinitionA set of observations produced by a device.

DefinitionA code that identifies what type of device observation this is.
Control1..1
TypeCodeableConcept

DefinitionIdentifiers assigned to this observation.
Control0..*

DeviceObservation.issued
DefinitionDate the measurements were made.
Control1..1
Typeinstant
RequirementsClinicians need to be able to check the date that the report was released.
AliasesDate Created; Date published; Date Issued
CommentsIn this context, it is assumed that the issued date is the date that the observations were actually made.
DeviceObservation.subject
DefinitionThe subject of the measurements. Usually, but not always, this is a patient. However devices are also used to make measurements on other things as well.

TypeResource(Patient|Group|Device)
