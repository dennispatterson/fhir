InvariantsDefined on this element
Inv-1: If the type is "Quantity", units must be provided (xpath: ((f:type/@value = 'Quantity') and (f:units)) or (not (f:type/@value != 'Quantity') and not (f:units)))
Inv-2: If the type is "Coding', system must be provided (xpath: (f:type/@value != 'Coding') or f:system)
Inv-3: If the type is "Quantity", ucum must be provided (xpath: ((f:type/@value = 'Quantity') and (f:ucum)) or ((f:type/@value != 'Quantity') and not (f:ucum)))
Inv-4: If the type is "SampledData", an sampling template must be provided (xpath: ((f:type/@value = 'SampledData') and (f:template)) or ((f:type/@value != 'SampledData') and not (f:template)))
